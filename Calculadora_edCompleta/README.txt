# Calculadora E.D. (LaTeX + Validación + Ejemplos)

## Render
Build Command: pip install -r requirements.txt
Start Command: gunicorn app:app

## Entrada amigable
- Potencias: x^2 (recomendado) o x**2
- Trig en español: sen(y) (se convierte a sin(y))
- Exponencial: e^x o e^(x+1)
- Multiplicación implícita: 300x^2y, xcos(y), ysen(x)
